🏏 IPL Analytics Dashboard (2008-2024)

📌 Overview
An end-to-end Data Analytics project analyzing 17 seasons of IPL cricket data from 2008 to 2024 using MySQL and Tableau.

🛠️ Tools & Technologies

MySQL — Database creation, data import & SQL analysis
Tableau Public — Interactive dashboard & visualizations
GitHub — Project documentation & version control


📊 Dashboards Built
Dashboard 1 — Team Performance

Most Wins by Team (Bar Chart)
Season Wins Analysis (Heat Map)
Toss Decision Analysis (Pie Chart)
Matches Played by City (Map)

Dashboard 2 — Player Performance

Top 10 Run Scorers (Treemap)
Top 10 Wicket Takers (Bar Chart)
Strike Rate vs Total Runs (Scatter Plot)
Player of the Match Awards (Bar Chart)


🔍 Key Insights

Mumbai Indians have the most wins in IPL history
Teams choosing to field first win more often
SK Raina is the highest run scorer in IPL history
AB de Villiers has the most Player of the Match awards
